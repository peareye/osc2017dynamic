Markdown Editor

When you edit any of the content, you will notice some interesting (different) characters that are typed before certain entries. This is a simple code called Markdown which when the entry is saved it is converted to HTML without a lot of the extra hidden code that more complex html editors create. This makes it easier to maintain clean html, and causes much less problems going forward.

An example of this is:

#### Check-in/Check-out

becomes

Check-in/Check-out

The four #'s convert the text after it into an H4 headline, like this

or

# H1 Title

becomes

H1 Title

For items that are part of a bulleted list you will see

    * Item 1
    * Item 2
    * Item 3

This will display like this:

  • Item 1
  • Item 2
  • Item 3
